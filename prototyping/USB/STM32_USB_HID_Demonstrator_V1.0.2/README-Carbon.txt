This directory contains the USB HID Demonstrator Windows application (UM0551)
provided by STMicroelectronics at the following URL:

    http://www.st.com/web/en/catalog/tools/PF257920

This application is designed to work with the STM32 USB-FS-Device development
kit (UM0424).

STMicroelectronics provides a PDF user manual and a ZIP file containing the
application software.  Both have been downloaded and/or expanded in this
directory.
