This directory contains STM3210E-EVAL demonstration firmware (UM0549) obtained
from the following URL:

    http://www.st.com/web/en/catalog/tools/PF257919

STMicroelectronics provides a PDF user manual and a ZIP file containing source
code.  Both have been downloaded and/or expanded in this directory.

The demonstration firmware includes a USB mass storage demo which runs on the
STM3210E-EVAL board.

The STM3210E-EVAL board uses the STM32F103ZGT6 processor.
